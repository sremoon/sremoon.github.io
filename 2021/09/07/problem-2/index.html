    题面

    你处于一根数轴上，初始在\(0\)号点，当你位于\(i\)时，你可以选择前往\(i - 1, i, i + 1\)中的任意一个，现在有\(T\)次攻击，每次攻击给定参数\(D_i, X_i\)，当你处于\(p\)时，若\(D_i = 0\)，表明攻击从\(X_i\)点由右向左射出，你将受到\(max(0,X_i - p)\)的伤害，而\(D_i = 1\)时，你将受到\(max(0, p - X_i)\)的伤害，试求出最小的受到的伤害

    题解

    确实不太会这种题

    我们先介绍一些概念

    我们称一个函数\(f\)在\(C\)上是（下）凸的，当且仅当

    \[\forall x_1, x_2 \in C,f(\frac{x_1 + x_2}{2}) \leq \frac{f(x_1) + f(x_2)}{2}\]

    为显简便，下文中省略\(C\)，凸函数有一些不错的性质

    • \(f, g\) 都是凸的时，对\(a, b > 0\)，\(af + bg\)是凸的

    \(af(\frac{x_1+x_2}{2}) + b g(\frac{x_1+x_2}{2}) \leq \frac{af(x_1)+bg(x_1) + af(x_2) + bg(x_2)}{2}\)

    • \(f, g\)都是凸的时，\(\max\{f, g\}\)是凸的

    \(\max \{f(\frac{x_1+x_2}{2}), g(\frac{x_1+x_2}{2})\} \leq \max\{\frac{f(x_1)+f(x_2)}{2}, \frac{g(x_1)+g(x_2)}{2}\} \leq \frac{\max\{f(x_1), g(x_1)\} + \max\{f(x_2), g(x_2)\}}{2}\)

    • \(f\)是凸的时，对于一个确定的常数\(L\)，记\(f_{max}(x) = max_{x-L\leq x_0 \leq x+L} f(x_0)\)，\(f_{min}(x) = min_{x-L\leq x_0 \leq x+L} f(x_0)\)，那么\(f_{max}(x)\)和\(f_{min}(x)\)在定义域内也是凸的

    凸函数一定是连续的，因此\(max\)和\(min\)都可以取到，下面的证明就感性一点吧...

    对于\(f_{max}\)，我们有\(f_{max}(x) = \max\{f(x-L), f(x+L)\}\)，因而是凸的

    对于\(f_{min}\)，考虑凸函数应该是一段降，一段常数，一段升的形状，对于这三段进行变形后，不难发现降的部分左移\(L\)，升的部分右移\(L\)，因而还是一个凸函数
