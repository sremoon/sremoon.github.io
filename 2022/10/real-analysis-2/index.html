Rudin的书写的深度不错

Lebesgue积分

Lebesgue积分的定义

  • 非负简单函数的积分

设\((X,\Gamma,\mu)\)是一个测度空间，\(S:X\to[0,+\infty]\)是一个可测简单函数，写为\(S = \sum_{i=1}^n \alpha_i 1_{A_i}\)

那么，定义\(S\)在\(X\)上的Lebesgue积分为\(\int_X S \text{d} \mu = \sum_{i=1}^n \alpha_i \mu(A_i)\)，对于任给的\(E \in \Gamma\)，定义\(S\)在\(E\)上的Lebesgue积分为\(\int_E S \text{d} \mu = \sum_{i=1}^n \alpha_i \mu(A_i \cap E)\)

  • 可测函数的积分

对测度空间\((X,\Gamma,\mu)\)，非负可测函数\(f:X\to[0,+\infty]\)，对\(E\in \Gamma\)，定义\(f\)在\(E\)上积分为

\(\int_E f \text{d} \mu = \sup \{\int_E S \text{d} \mu : 0 \leq S \leq f, S \;is\;simple\;and\;measurable\}\)

对于一般的可测函数\(f\)，我们将其写为\(f=f^+-f^-\)，如果\(\int_E f^+ \text{d} \mu\)和\(\int_E f^{-} \text{d} \mu\)中至少有一个是有限的，我们就定义\(\int_E f\text{d}\mu = \int_E f^+ \text{d} \mu-\int_E f^{-} \text{d} \mu\)，否则，我们称\(f\)的积分无定义

接下来介绍Lebesgue积分的一些性质，设\((X,\Gamma,\mu)\)为测度空间，\(E,F\)为可测集，\(f,g\)为非负可测函数

  • 如果\(0 \leq f \leq g\)，那么\(\int_E f \text{d} \mu \leq \int_E g\text{d} \mu\)
  • 如果\(E \sub F\)，那么\(\int_E f \;\text{d}\mu \leq \int_F f\;\text{d} \mu\)
  • 如果\(\alpha \in (0,+\infty)\)，那么\(\int_E \alpha f\text{d} \mu = \alpha \int_E f\text{d}\mu\)
  • 如果\(f|_E = 0\)或者\(\mu(E)=0\)，那么\(\int_E f \text{d} \mu = 0\)
  • 如果\(E \sub X\)，那么\(\int_E f \text{d} \mu = \int_X f 1_{E} \text{d} \mu\)

Lebesgue Monotone convergence Theorem

单调收敛定理：设\(\{f_n\}\)为\(X\)上的可测函数列，并且\(0 \leq f_1 \leq f_2 ... \leq f_n \leq ... \leq +\infty\)，并且\(\forall x \in X, \lim_{n \to \infty} f_n(x) = f(x)\)，那么\(f\)非负可测，并且\(\lim_{n \to \infty} \int_X f_n \text{d} \mu = \int_X f\text{d}\mu\)

证明：\(f\)非负可测可以根据我们之前的讨论结果得出

我们用到如下的引理

如果\(S\)是一个简单函数，那么\(\varphi(E):=\int_E S\text{d}\mu\)是\(\Gamma\)上的测度

首先，\(\int_X f_1 \text{d} \mu \leq ... \int_X f_n \text{d} \mu \leq...\leq \int_X f \text{d} \mu\)，因此\(\lim \int_X f_n \text{d} \mu \leq \int_X f \text{d} \mu\)

下面证明\(\int_X f\text{d} \mu \leq \lim \int_X f_n \text{d} \mu\)，只需证明对满足\(0 \leq S \leq f\)的简单函数\(S\)，有\(\int_X S \text{d} \mu \leq \lim \int_X f_n \text{d} \mu\)

固定\(C \in (0,1)\)，设\(E_n:=\{f_n(x)>C\cdot S(x)\}\)，那么\(E_n\)可测，\(E_n \sub E_{n+1}\)，并且\(X = \bigcup E_n\)

因此\(\int_X f_n \geq \int_{E_n} f_n \geq \int_{E_n} C\cdot S=C\int_{E_n} S\)，令\(n \to \infty\)，注意到\(\int_E S\)是一个测度，因此\(\lim C \int_{E_n} S = C \lim \varphi(E_n)=C \varphi(\bigcup E_n) = C \varphi(X)\)，最终得到\(\lim \int_X f_n \geq C \varphi(X)\)，由于\(C\)在\((0,1)\)中任取，最后又\(\lim f_X f_n \geq \varphi(X)=\int_X S\)，证毕

我们使用Lebesgue Monotone convergence Theorem来得到Lebesgue积分的可加性

如果\(f,g\)为非负可测函数，那么\(\int (f+g)=\int f + \int g\)

根据之前的简单函数逼近的讨论，我们知道存在递增简单函数列\(\{S_n\},\{T_n\}\)逼近\(f,g\)，再根据下面的引理

如果\(S,T\)为可测简单函数，那么\(\int (S+T)=\int S + \int T\)

自然\(\{S_n+T_n\}\)逼近\(f+g\)，那么根据单调收敛定理，我们有\(\int (f+g)=\lim \int (S_n+T_n) = \lim \int S_n + \lim \int T_n = \int f + \int g\)

级数的收敛自然是单调收敛定理的一个情形，因此有

设非负函数\(\{f_n\}\)可测，\(f = \sum_{n=1}^{\infty} f_n\)，那么\(\int_X f \text{d}\mu = \sum_{n=1}^{\infty} \int_X f_n \text{d} \mu\)

后者的证明用到了有限和与积分可以交换顺序的性质

(Fatou's lemma) 设非负函数族\(\{f_n\}\)可测，那么\(\int_X \lim \inf f_n d\mu \leq \lim \inf \int_X f_n d\mu\)

积分的性质

接下来，我们将考虑所有的函数，而不仅限于非负函数

  • 设\(f\)可测，如果\(\int_X |f| d\mu < \infty\)，则称\(f\)为Lebesgue可积

    Lebesgue可积函数的全体记为\(L^1(X,\mu)\)

  • 换元：设\(f:X\to[0,\infty]\)可测，\(\varphi:\Gamma \to[0,\infty], \varphi(E) = \int_E f d \mu\)

    那么，\(\varphi\)是\((X,\Gamma)\)上的正测度，并且\(\forall g:X\to[0,\infty]\)，\(\int_X g d \varphi = \int_X gf d \mu\)

    在这个换元的理解下，\(\mu(E) = \int_E 1 d\mu\)

  • 替换积分域：\(\int_E f d\mu = \int_X f1_{E} d\mu\)

    特别的，\(\int_B f = \int_A f + \int_{B/A} f\)

  • 绝对值的积分：如果\(f \in L^1(X,\mu)\)，那么\(|\int_X f d\mu| \leq \int_X |f| d\mu\)

  • 积分的线性性：如果\(f,g \in L^1(X,\mu)\)，那么\(\alpha f + \beta g \in L^1(X,\mu)\)，并且\(\int_X \alpha f + \beta g = \alpha \int_X f + \beta \int_X g\)

Lebesgue Dominated convergence Theorem

控制收敛定理：设\(\{f_n\}\)可测，如果\(f_n \to f\)，且存在\(g \in L^1(X,\mu)\)使得\(|f_n| \leq g\)，那么\(f \in L^1(X,\mu)\)，并且\(\lim \int_X f_n d \mu = \int_X f d \mu\)

注意到\(2g - |f_n - f|\)非负可测

应用Fatou's lemma得到\(\lim\inf \int 2g-|f_n-f| \geq \int \lim \inf 2g-|f_n-f| = \int 2g\)

这告诉我们\(\lim \int |f_n - f| = 0\)，而\(|\lim \int f_n - \int f| = |\lim \int(f_n-f)|= \lim |\int (f_n-f)| \leq \lim \int |f_n-f|\)

如果\(f \in L^1(X,\mu)\)，那么\(\varphi(E):=\int_E f\text{d}\mu\)是\(\Gamma\)上的实测度

类似于之前证明正测度，不过我们将使用控制收敛定理而不是单调收敛定理

（积分的绝对连续性）：如果\(f \in L^1(X,\mu)\)，那么\(\forall \epsilon>0, \exists \delta>0\)，对\(\mu(E)<\delta\)的\(E\)，有\(\int_E |f| d\mu < \epsilon\)

考虑取简单函数列\(\{S_n\}\)去逼近\(f\)，\(S_n\)有界，而\(n\)足够大时，\(f-S_n\)根据收敛性也足够小，从而得到一个证明

其余讨论

  • almost everywhere 以及almost all的含义都是除了一个零测集外成立，一般根据适用语境的不同会有所调整

  • 设\((X,\Gamma,\mu)\)是测度空间，且\(f,g\)为可测函数，如果\(\mu{\{f(x)\neq g(x)\}}=0\)，则称\(f,g\)几乎处处相等，记为\(f \sim g\)，这是可测函数上的一种等价关系

对可测函数\(f,g\)，如果\(f \sim g\)，那么\(f^+ \sim g^+, f^- \sim g^-, |f| \sim |g|\)

易证

如果\(f \sim g\)，并且$f \(存在，那么\)g\(也存在，并且\)g = f$

这告诉我们，在积分的意义下，零测集没有讨论的意义，举个例子：

设\(\{f_n\}:X\to[-\infty,+\infty]\)可测，且\(\sum \int_X |f_n| d \mu < +\infty\)，那么

记\(f(x) = \sum f_n(x)\)，对a.a. \(x \in X\)，\(f(x)\)收敛

则\(f \in L^1(X,\mu)\)，且\(\int_X f d \mu = \sum_{n=1}^{\infty} \int_X f_n d \mu\)

证明：记\(g = \sum |f_n|\)，那么\(g\)可测，并且\(\int g = \int \sum |f_n| = \sum \int |f_n| < \infty\)（单调收敛定理）

记\(E = \{g(x)<\infty\}\)，由于\(g \in L^1(X,\mu)\)，那么\(\mu(E^c) = 0\)

从而\(\forall x \in E, g(x)=\sum |f_n(x)| < \infty\)，故\(f(x)=\sum f_n(x)\)在\(E\)上收敛

注意到在\(E\)上，\(|f| \leq g < \infty\)，根据控制收敛定理，\(f \in L^1(X,\mu)\)，\(\int_E \sum f_n(x) = \int_E f d\mu\)

设\((X,\Gamma,\mu)\)为测度空间，定义\(\Gamma^* = \{E:\exists A,B \in \Gamma, A \sub E \sub B, \mu(B/A)=0\}\)，那么\(\Gamma^*\)为\(\sigma-\)代数

如果在这里补充定义\(\mu(E)=\mu(A)\)，那么\((X,\Gamma^*,\mu)\)是完备化的测度空间

并且，对\(\mu(E)=0\)的集合，如果\(F \sub E\)，那么\(F \in \Gamma^*\)，且\(\mu(F)=0\)

这可以对零测度进行一个补充定义

对于完备化的测度空间，修改函数在零测集上的函数值不改变该函数的积分

完备化的主要目的在于确保修改定义之后的函数是一个可测函数

  • 如果\(f:X\to [0,\infty]\)可测，\(\int_E f d \mu = 0\)，那么\(f\)在\(E\)上几乎处处为\(0\)
  • 如果\(f \in L^1(X, \mu)\)并且\(\forall E, \int_E f d \mu = 0\)，那么\(f\)在\(X\)上几乎处处为\(0\)
  • 如果\(f \in L^1(X, \mu)\)且\(|\int_X f d \mu| = \int_X |f| d\mu\)，那么\(f \geq 0\)或者\(f \leq 0\)在\(X\)上几乎处处成立

收敛

  • Def：对于测度空间\((X,\Gamma, \mu)\)，\(\{f_n\}\)可测，我们有以下几种收敛
    • 处处收敛：\(\forall x, f_n(x) \to f(x)\)
    • 几乎处处收敛：\(\mu(\{f_n(x) \not\to f(x)\}) = 0\)
    • 一致收敛：\(\forall \epsilon > 0, \exists N, \forall n>N, |f_n(x)-f(x)| < \epsilon\)
    • \(L^1\)收敛：当\(\{f_n\},f \in L^1(X,\mu)\)时，\(f_n \to f\)以\(L^1\)收敛当且仅当\(\int_X |f_n-f|\to 0\)）（实际上也就是以\(L^1\)范数收敛）
    • 依测度收敛：\(\forall \epsilon, \mu(\{|f_n(x)-f(x)|>\epsilon\})\to 0\)
    • 近一致收敛：\(\forall \epsilon, \exist \mu(E)<\epsilon\)，使\(f_n \to f\)在\(X \setminus E\)上一致收敛

下面我们来逐步介绍这些收敛之间的强弱关系

s1: 如果\(\{f_n\}\)依\(L^1\)收敛，那么\(\{f_n\}\)依测度收敛

证明：\(\forall \epsilon > 0\)，如果\(f_n \to f\)以\(L^1\)收敛，那么\(\int_X|f_n-f| \to 0\)

注意到\(\mu(\{|f(x)|>\alpha\}) \leq \frac{1}{\alpha} ||f||_1\)，因此\(\mu(\{|f_n-f| > \epsilon\}) \leq \frac{1}{\epsilon} ||f_n-f||_1 \to 0\)

s2: 如果\(\mu(X)<+\infty\)并且\(f_n \to f\)几乎处处收敛，那么\(f_n \to f\)依测度收敛

对应到概率论中，这实际上对应以概率\(1\)收敛和依概率收敛

证明：不难知道，\(f_n(x) \to f(x)\)当且仅当\(\lim \sup |f_n-f|=0\)

因此$({f_n(x)f(x)}) = 0 (_{m=1}^{} {|f_n - f| > }) = 0 $

这又等价于说明\(\mu(\{\lim \sup |f_n - f| > \frac{1}{m}\}) = 0\)

记\(F_n = \bigcup_{k \geq n} \{|f_k - f| > \frac{1}{m}\}\)，自然\(X \supset F_1 \supset F_2 ...\)，由于\(\mu(X)<+\infty\)，因此\(\lim \mu(F_n) = \mu(\lim F_n) = 0\)，而\(\mu(\lim F_n) = 0\)等价于\(\mu( \{\lim \sup |f_n - f| > \frac{1}{m}\}) = 0\)，这就是我们想证的

s3: 设\(\{\epsilon_n\}\)单调递减收敛至\(0\)，\(\{g_n\}\)可测，令\(E_n = \{|g_n(x)|>\epsilon_n\}\)，如果\(\sum \mu(E_n) < \infty\)，那么\(g_n\)几乎处处收敛到\(0\)，同时，\(g_n\)近一致收敛到\(0\)

证明：要说明\(g_n\)几乎处处收敛到\(0\)，我们只需要说明\(\mu(\lim \sup |g_n| > \frac{1}{m})=0\)

由于\(\{\epsilon_n\}\)单调递减收敛至\(0\)，这其实只需要说明\(\mu(\lim \sup |g_n| > \epsilon) = 0\)，由下面这个引理可以直接给出

  • 如果\(\sum \mu(E_n) < \infty\)，那么\(\mu(\lim \sup E_n) = 0\)

说明\(g_n\)近一致收敛，当\(N\)足够大时，\(\sum_{n = N} \mu(E_n) < \epsilon\)，取\(E = \bigcup {E_n}\)，在\(X \setminus E\)上，\(g_n\)处处收敛到\(0\)

s4: 如果\(\{f_n\}\)依测度收敛到\(f\)，那么存在子列\(\{f_{n_k}\}\)，使得\(\{f_{n_{k}}\}\)几乎处处收敛并且近一致收敛到\(f\)

证明：联想到上一条定理，我们构造\(\epsilon_n = \frac{1}{n}, E_m = \mu(\{|f_{n_m}-f| > \epsilon_m\}) < \frac{1}{2^m}\)即可，\(n_m\)可够取得则根据依测度收敛的性质

Theorem：如果\(\mu(X) < \infty\)并且\(f_n \to f\)几乎处处收敛，那么\(f_n\to f\)近一致收敛

证明：令\(g_n = \sup_{k \geq n} f_k\)，那么\(\lim g_n = \lim \sup f_n = \lim f_n = f\)，对几乎所有的\(x\)，因此\(g_n \to f\)几乎处处收敛，根据s2，\(g_n\)依概率收敛，根据s4，存在子列\(\{g_{n_k}\}\)几乎处处收敛

注意到\(|f_n - f| \geq g_{n_k}, \forall n \geq n_k\)，因此当\(g_{n_k}\)收敛到\(0\)时，\(|f_n-f|\)也收敛到\(0\)，因此\(f_n\to f\)近一致收敛

Theorem：如果\(f_n \to f\)近一致收敛，那么\(f_n \to f\)几乎处处并且依测度收敛

证明：\(f_n\to f\)近一致收敛，因此对\(\epsilon > 0\)，存在\(\mu(E_{\epsilon})<0\)，在\(X \setminus E_{\epsilon}\)上处处收敛

注意到\(\{f_n(x) \not \to f(x)\} \sub E_{\epsilon}\)，因此\(\mu\{f_n(x) \not \to f(x)\} \leq \mu\{E_{\epsilon}\} \leq \epsilon\)，将\(\epsilon \to 0\)，我们就得到了\(f_n\to f\)几乎处处收敛。固定\(c\)，当\(n\)足够大时，\(\{|f_n(x)-f(x)|\geq c\} \sub E_{\epsilon}\)，此时和上面一样，有\(\lim \mu\{|f_n(x)-f(x)|\geq c\} = 0\)，这就证明了依测度收敛

Lebesgue测度

  • Def：我们称\(I=\{x:a_i<x_i<b_i\}\)为一个开矩体，记其体积为\(vol(I) = \prod(b_i - a_i)\)
  • Def：如果\(E \sub \bigcup I_k\)（可数个），其中\(I_k\)为开矩体，那么称\(\{I_k\}\)为\(E\)的一个\(L-\)覆盖，称\(m^*(E) := \inf \{ \sum_{k=1}^{\infty} vol(I_k) : E \sub \bigcup I_k\}\)为\(E\)的外测度

外测度满足一些类似于测度的性质：

  • \(m^*(E) \geq 0\)且\(m^*(\emptyset) = 0\)
  • 如果\(E_1 \sub E_2\)，那么\(m^*(E_1) \leq m^*(E_2)\)
  • \(m^*(\bigcup E_k) \leq \sum m^*(E_k)\)（可列次可加性）

外测度还需要有一些看起来很正确的性质：

对于可数点集\(A\)，\(m^*(A) = 0\)

证明一个点是0之后，运用可列次可加性

对于开矩体\(I\)，\(m^*(\bar{I}) = vol(I)\)，其中\(\bar{I}\)为\(I\)的闭包

运用紧集的有限子覆盖性质进行夹逼

对于超平面\(H\)，\(m^*(H) = 0\)

超平面虽然是无界的，但我们可以用可数个有界的外测度和为0的开矩体集进行放缩

对于开矩体\(I\)，\(m^*(I)=vol(I)\)

开矩体就是其闭包去除若干超平面

如果\(d(E_1,E_2) > 0\)，那么\(m^*(E_1\cup E_2) = m^*(E_1) + m^*(E_2)\)

我们通过控制覆盖\(E_1 \cup E_2\)开矩体集中的开矩体足够小，来保证其不同时与\(E_1,E_2\)交，之后就方便放缩了

\(m^*(E+\{x_0\})=m^*(E)\)

不难证明
